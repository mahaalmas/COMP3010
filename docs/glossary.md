# Glossary of Important Terms

**CloudTrail** – AWS service that logs administrative actions.  
**S3 ACL** – Access Control List that governs bucket permissions.  
**MFA** – Multi-Factor Authentication.  
**PutBucketAcl** – API call to modify S3 bucket permissions.  
**WinHostMon** – Splunk sourcetype for Windows host monitoring.  
**Processor Identifier** – CPU model string from hardware logs.  
**REST.PUT.OBJECT** – S3 operation for uploading a file.  
**FQDN** – Fully Qualified Domain Name of a host.
